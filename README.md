## Getting Started

### Setup

> **Note**
> Do the Server-side Setup first

Server-side:

```sh
cd <Project_Dir>/server
npm i
```

Front-end:

```sh
cd <Project_Dir>/project-manager
npm i
```

### Running

Run the whole Project:

```sh
cd <Project_Directory>/server
npm run dev
```

Just Front-end: (not recomended)

```sh
cd <Project_Dir>/project-manager
npm start dev
```

Just Server-side:

```sh
cd <Project_Dir>/server
npm run server
```
